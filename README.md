# Yacoub's Subs

Multi-source subtitle addon for Kodi with extensive language support.

## Features

- 40+ languages supported
- 12 providers including OpenSubtitles, SubDL, SubSource, and more
- Movies, TV Shows, and Anime support
- User-configurable API credentials

## Installation

Add the repository to Kodi:
https://yacoubs00.github.io/

Then install "Yacoub's Subs" from the repository.

## Configuration

Configure your API keys in addon settings for:
- OpenSubtitles
- SubDL
- SubSource

Other providers work without configuration.
