WHY THIS HAPPENS:

Kodi has TWO separate texture caches:
1. Addon detail view (this works - you see icon in settings)
2. Addons list view (this is cached and NOT updating)

The addons list cache is VERY persistent on Xbox and doesn't 
automatically refresh when you:
- Update the addon
- Restart Kodi
- Change versions

SOLUTION - CLEAR XBOX KODI TEXTURE CACHE:

Method 1: Delete Texture Database (RECOMMENDED)
   1. Exit Kodi completely
   2. Navigate to: Q:\XBOX APPS\KODI\Database\
   3. Find file: Textures13.db
   4. DELETE this file
   5. Start Kodi
   6. Icon should now show in addons list

Method 2: Clear Thumbnails Folder
   1. Exit Kodi completely
   2. Navigate to: Q:\XBOX APPS\KODI\Thumbnails\
   3. DELETE entire Thumbnails folder
   4. Start Kodi
   5. Kodi will regenerate all thumbnails

Method 3: File Manager in Kodi
   1. Settings â†’ File Manager
   2. Add source â†’ Browse
   3. Navigate to Profile Directory
   4. Delete Database/Textures13.db
